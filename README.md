# Cura Termbox FFI Adapter

[Cura][cura] adapter for [Termbox][termbox] using [rb_termbox][rb_termbox].

## Install

### Bundler: `gem 'cura-adapter-termbox-ffi'`

### RubyGems: `gem install cura-adapter-termbox-ffi`
